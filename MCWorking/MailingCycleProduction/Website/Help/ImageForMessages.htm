Including an image in the message

Uploading an image from local machine

Click on the "Image" button () inside the Message text tool bar. The file upload option becomes visible. Browse for the file using the browse button and click on the "Upload Image" button.

To insert an image into message

1. Click on the "Image" button () inside the Message text tool bar. The list of images appear as a list above the message text box.
2. Click on the "select" link corresponding to the image that is required.
3. Select the desired alignment, border width for image and margin for the image in the message text tool bar.
4. Select the desired location of the image in the message.
5. Click on the "Insert Image" button in the message text tool bar.
